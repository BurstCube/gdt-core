*********************************************
The Spectra Package (:mod:`gdt.core.spectra`)
*********************************************
The ``spectra`` package provides classes for performing spectral fitting, along
with a built-in list of spectral functions that can be used in the fitting. 
There are also classes provided that represent spectral fit results and metadata.
